# תוכנית יישום: מערכת ניהול מועמדויות מאוחדת

## סקירה כללית

תוכנית יישום למערכת ניהול מועמדויות מאוחדת עבור הנהלת בתי המשפט.
הטכנולוגיות: Angular (Frontend) + .NET Core (Backend API) + Entity Framework Core + SQL Server.
המשימות מסודרות בצורה אינקרמנטלית כך שכל שלב בונה על הקודם.
